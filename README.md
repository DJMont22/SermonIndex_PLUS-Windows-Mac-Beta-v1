# SermonIndex +
A desktop study and download companion for SermonIndex content.

## Overview
SermonIndex + is a Windows desktop app(Mac Coming once new API is implamented) built to help users browse, read, preview, organize, create and package content to a zip made available through the official SermonIndex website and API. It is designed for studying sermons, books, and Bible content in one place, with tools for reading, downloading, collecting notes, and preparing study packages.

This app is intended to work from official SermonIndex sources only. SermonIndex states that its audio sermons, video sermons, and text sermons are presented in their original form, and that its commentaries and Bibles come from open-source locations and original writings of translators and authors. 

## What the program does
The app lets you:
- browse speakers from SermonIndex
- load sermons and books by speaker
- read sermon or book content in the built-in reader
- use the Bible viewer to open translations, chapters, and verses
- jump directly to a Bible reference
- compare Bible passages with a parallel translation view
- listen to available audio inside the app
- download available files exposed by SermonIndex
- send selected text into the Index Study workspace via right click or copy paste to tab.
- build study notes and package selected resources into a ZIP file

## Official SermonIndex content used by the app
This program is designed to use official SermonIndex content only.
To have a Web app designed for creation and study.

SermonIndex publicly presents:
- audio sermons
- video sermons ( coming in next update )
- text sermons
- books
- online Bibles
- downloadable sermon media and text files such as MP3, MP4, PDF, and TXT where available. 

**SermonIndex also states that its materials are offered freely, and the site’s FAQ explains that its sermons and materials may be copied for personal use and distribution.**


## How audio, pictures, and text are used
### Audio
When SermonIndex provides audio for a sermon or Bible resource, the app can use that official audio link for in-app playback and download. SermonIndex’s own support pages explain that audio sermons are downloadable as MP3 files, and video sermons may be available as MP4 files. 

### Pictures
If official SermonIndex content provides a speaker image, cover image, or other image attached to a sermon, speaker, or reading resource, the app can display that image in the reader. The program is not intended to invent or scrape unrelated media; it only shows image content that is already exposed by the official source.

### Text
The reader and Bible viewer use official SermonIndex-provided text content where available. SermonIndex notes that its audio, video( coming soon can possibly download video just cant view in app yet), and text sermons are kept in their original form, while some transcripts, summaries, and scripture references on the site may involve AI-assisted generation or scripting on the SermonIndex side.

## Main sections of the app
### 1. Main tab
Use the main tab to browse SermonIndex content.

Main features include:
- speaker list
- speaker search
- sermon list
- book list
- built-in reader
- Bible viewer
- package queue

### 2. Reader
The reader is where sermon, book, and Bible content is displayed. Depending on the resource, it can show:
- title and reference information
- reading text
- official image content when available
- in-app audio controls when official audio is available

### 3. Bible Viewer
The Bible viewer is for reading and navigating online Bible content made available through SermonIndex. The site lists multiple online Bible translations, including examples such as Berean Standard Bible, King James Version, World English Bible, Darby Translation, Geneva Bible 1599, Young’s Literal Translation, Free Bible Version, and others.

Bible viewer features include:
- translation selection
- book, chapter, and verse entry
- jump-to reference
- previous / next chapter
- previous / next book
- optional parallel translation view ( ch & vs selection is linked to parallel and main translation
  (find bible chapter and book then click parallel and select your translation to pop open at bottom. click parallel again to make it go away.)
- send selected text to Index Study

### 4. Index Study tab
This tab is for building personal study notes from content selected in the app. You can:
- receive text sent from the reader
- write sermon notes
- create Bible studies
- format your study content
- export or package your study with selected resources

### 5. Package Queue
The package queue is used to collect resources you want to bundle together. This is useful for preparing sermon study packs, Bible study collections, or teaching bundles. ( this can be removed or sized to a seperate window for more viewing space same with activity log * Recommended * )

## How to use the app
### Browse a speaker
1. Open the app.
2. Go to the main content area.
3. Search for a speaker or choose one from the speaker list.
4. Click the speaker to load sermons and books.

### Open a sermon or book
1. Select a loaded speaker.
2. Click a sermon or book entry.
3. The selected resource opens in the reader.

### Use the Bible viewer
1. Switch to the Bible viewer.
2. Pick a translation.
3. Enter a book, chapter, and verse if needed.
4. Press **Jump To** or hit Enter.
5. Use **Prev Book**, **Prev Ch**, **Next Ch**, or **Next Book** to continue reading.

### Use parallel view
1. Load a Bible passage in the main Bible viewer.
2. Click the **Parallel** button.
3. Choose the comparison translation.
4. The main passage and the parallel passage appear in separate reading areas.
5. Click **Parallel** again to hide it.

### Send text to Index Study
1. Highlight text in the reader.
2. Use the send-to-study option.
3. The selected content is inserted into the Index Study tab for note building.

### Download content
If a resource has an official downloadable file, use the built-in download option. SermonIndex’s own support pages note that downloadable content may include MP3 for audio, MP4 for video, and PDF or TXT for text-based media where available.

### Listen to audio
If official audio is available for the selected resource, use the in-app audio controls in the reader. The app is intended to use official SermonIndex audio sources only.

## Installation
This app is normally distributed through a Windows installer.

Typical install process:
1. Run the installer EXE.
2. Choose the install location.
3. Select optional shortcuts if offered.
4. Finish installation.
5. Launch **SermonIndex +** from the desktop or Start Menu.

The installer places the application files in **Program Files\SermonIndex +** unless you choose another location.

## Uninstall
The installed application includes an uninstaller.
Within Program files. click uninstall and follow for complete uninstall.

Typical uninstall process:
1. Open Windows **Installed Apps / Apps & Features**, or
2. Open the program’s install folder and run **Uninstall.exe**, or
3. Use the Start Menu uninstall entry if available.

This removes the installed app files and shortcuts created by the installer.
